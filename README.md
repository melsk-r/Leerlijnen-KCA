# Leerlijnen Kennis Centrum Architectuur

In deze repository worden de leerlijnen beschreven benodigd voor het werkzaam kunnen zijn binnen het Kennis Centrum Architectuur (KCA).
Vooralsnog betreft het alleen de leerlijn voor Informatieanalist en gegevensarchitect, in de wandelgangen StUF-beheerder genoemd.

Per leerlijn wordt de benodigde kennis vastgelegd, op welke termijn men over deze kennis (vanaf aanvang van de werkrelatie) moet kunnen beschikken en door welke partij deze kennis gedoceerd wordt.
