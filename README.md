# Leerlijnen Kennis Centrum Architectuur

In deze repository worden de leerlijnen beschreven benodigd voor het werkzaam kunnen zijn binnen het Kennis Centrum Architectuur (KCA) van VNG Realisatie.
Vooralsnog betreft het alleen de leerlijn voor Informatieanalist en Gegevensarchitect, in de wandelgangen StUF-beheerder genoemd.

Per leerlijn wordt de benodigde kennis vastgelegd, op welke termijn men over deze kennis (vanaf aanvang van de werkrelatie) moet kunnen beschikken en door welke partij deze kennis gedoceerd wordt.

## Website

Deze Hugo website wordt automatisch gedeployed naar GitHub Pages via GitHub Actions.

### Lokale ontwikkeling

1. Clone de repository met submodules:
   ```bash
   git clone --recurse-submodules https://github.com/VNG-Realisatie/Leerlijnen-KCA.git
   ```

2. Start de Hugo development server:
   ```bash
   hugo server -D
   ```

3. Ga naar `http://localhost:1313` om de website te bekijken.

### Deployment

De website wordt automatisch gedeployed naar [https://VNG-Realisatie.github.io/Leerlijnen-KCA/](https://VNG-Realisatie.github.io/Leerlijnen-KCA/) bij elke push naar de main branch.
